## English

### Overview
**Wheel of Your Fortune** is a premium, interactive web application built with React and Vite. It allows users to create a custom list of options and let a sleek, animated wheel decide their fate. Designed with a modern "glassmorphism" aesthetic, it features smooth transitions, vibrant gradients, and a responsive layout.

[![Vite](https://img.shields.io/badge/Vite-B73BFE?style=for-the-badge&logo=vite&logoColor=FFD62E)](https://vitejs.dev/)
[![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=for-the-badge&logo=typescript&logoColor=white)](https://www.typescriptlang.org/)

### Key Features
- **Interactive SVG Wheel**: Smooth physics-based spinning animation.
- **Dynamic Management**: Easily add or remove options in real-time.
- **Modern UI**: Dark mode, glass-style containers, and vibrant peach-to-pink gradients.
- **Responsive**: Fully optimized for both desktop and mobile devices.
- **Winner Announcement**: Automatic result overlay when the wheel stops.

### Tech Stack
- **Frontend**: React 19 (Hooks, Functional Components)
- **Styling**: Vanilla CSS with modern variables and animations
- **Build Tool**: Vite (Fast HMR)
- **Language**: TypeScript (Type safety)

### Getting Started

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd wheelspin/wheelspiner
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

## Türkçe

### Genel Bakış
**Wheel of Your Fortune**, React ve Vite ile geliştirilmiş, premium ve interaktif bir şans çarkı uygulamasıdır. Kullanıcıların kendi seçenek listelerini oluşturmalarına ve şık, animasyonlu bir çark ile karar vermelerine olanak tanır. Modern "glassmorphism" (cam efekti) estetiği ile tasarlanmış olup yumuşak geçişler, canlı degradeler ve duyarlı (responsive) bir düzen sunar.

### Temel Özellikler
- **İnteraktif SVG Çark**: Fizik tabanlı, yumuşak dönme animasyonu.
- **Dinamik Yönetim**: Gerçek zamanlı olarak seçenek ekleme ve çıkarma.
- **Modern Arayüz**: Karanlık mod, cam efekti kaplar ve canlı şeftali tonlarında degradeler.
- **Duyarlı Tasarım**: Masaüstü ve mobil cihazlar için tam uyumlu.
- **Kazanan Duyurusu**: Çark durduğunda otomatik kazanan ekranı.

### Teknoloji Yığını
- **Frontend**: React 19
- **Stil**: Modern CSS değişkenleri ve animasyonları
- **Derleme Aracı**: Vite
- **Dil**: TypeScript

   ```bash

   ```

